Quick Start

Install & Run

1. Install text-injection dependency (Debian/Ubuntu)

sudo apt install ydotool

2. Make the downloaded app executable

chmod +x LocalVoice-Linux

3. Run the application

./LocalVoice-Linux

After Installation

Launch LocalVoice from your terminal or application menu:

localvoice
Uninstall & Free Space

To completely remove the application and the 1.5GB cached AI models, run these commands:

rm LocalVoice-Linux
rm -rf ~/.cache/huggingface/hub/models--Systran--faster-whisper-large-v3-turbo
Install & Run

No terminal needed! Just download LocalVoice.exe and double-click to run.

Uninstall

Delete LocalVoice.exe to remove the app.

To free up disk space, delete the AI model cache folder at:
C:\Users\YOUR_NAME\.cache\huggingface
